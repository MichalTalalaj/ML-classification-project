# ML Classification Project

A machine learning classification project implementing and comparing multiple classification models using Jupyter.

## Project Overview

This repository includes a complete workflow for a classification task, from data loading and preprocessing to model training and evaluation. The goal of the project is to explore different classifiers, compare their performance, and visualize results.

## Project Contents

- `code.ipynb` — main notebook with the full classification pipeline  
- `artifical_train_data.csv`, `artifical_test_data.csv` — synthetic dataset used for training and testing  
- `artifical_train_labels.csv` — target labels for training  
- `model_comparison.png` — visual comparison of model performance  
- `report.pdf` — written summary of objectives, methods, and results  
- `project_task.pdf` — description of the task or project requirements  

## Classification Workflow

1. **Data Loading and Exploration**  
   - Import datasets and analyze structure and features.  
2. **Preprocessing**  
   - Handle missing values, scale or transform features as needed.  
3. **Model Training and Comparison**  
   - Train multiple classification algorithms.  
   - Compare performance using relevant metrics.  
4. **Visualization**  
   - Generate comparison charts and performance plots.
